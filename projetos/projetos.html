Hashtag Programação - Intensivão de Javascript

Este evento foi uma imersão completa e intensiva, estruturada em 4 aulas práticas, com o objetivo de equipar desenvolvedores e iniciantes com as habilidades mais demandadas do mercado, utilizando o JavaScript. O curso foi desenhado para ir além da teoria, focando em uma abordagem 100% prática para que você pudesse criar aplicações completas do zero.

Projetos

Hash-AudioBook

O projeto é a criação de um player de audiobook, onde os usuários podem ouvir "Dom Casmurro", navegar entre capítulos e controlar a reprodução, demonstrando a interação prática de HTML, CSS e JavaScript.

|

Apple Watch

Utilizando JavaScript, buscamos proporcionar uma experiência dinâmica e acessível, permitindo que os usuários visualizem as diferentes combinações de cor da pulseira e tamanho do Apple Watch em tempo real.

|

Cardapio Digital

No Cardápio Digital, exploramos a implementação da lógica utilizando React. O objetivo é facilitar a interação do usuário com o uso do React, permitindo uma navegação intuitiva entre as opções do menu.

|

Chatbot com IA

Neste projeto você pode conversar em tempo real com uma IA. O projeto foi totalmente feito usando a IA do Lovable, em uma interface moderna feita em TypeScript
